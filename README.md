# Project VHDL Waching Machine

A digital logic design of an automatic washing machine controller implemented in VHDL.
- **FSM**
- **Timer Integration**
- **Sensor Simulation**
- **Status Indicators**



## Development Environment
- **Software:** Xilinx ISE 14.7 (Project Navigator)
- **Simulator:** ISim (Integrated Simulation Environment)
- **Language:** VHDL-93


## How to start?
1. Open `utt.xise` in Xilinx ISE.
2. Select `test.vhd` in **Simulation** view.
3. Run **Simulate Behavioral Model** in the Processes pane.
4. Load `wyniki.wcfg` to view pre-configured waveforms.
